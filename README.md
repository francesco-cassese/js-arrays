# JS Arrays

## 📌 Descrizione
Esercizio pratico dedicato alla manipolazione degli array in JavaScript. Il focus è l'elaborazione di liste di dati attraverso l'uso di cicli iterativi per risolvere task specifici su diversi script.

## 🎯 Obiettivo
Approfondire la conoscenza delle strutture dati lineari (Array) e padroneggiare l'iterazione manuale per leggere, modificare o filtrare i dati contenuti.

### Focus Tecnico:
- **Array Traversal:** Accesso agli elementi tramite indice.
- **Debugging:** Utilizzo sistematico dei *DevTools* del browser e `console.log` per la verifica immediata dei risultati di ogni task.
- **Proprietà .length:** Gestione dinamica dei cicli in base alla dimensione dell'array.

## 🛠️ Tecnologie Utilizzate
- **HTML5:** File di entry point per l'esecuzione degli script.
- **JavaScript (ES6):** Logica di manipolazione dati e cicli iterativi.
